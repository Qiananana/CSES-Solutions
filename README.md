## 🧠 Strategic Logic Board

| Problem | Category | Key Strategic Insight | Complexity |
| :--- | :--- | :--- | :--- |
| **Weird Algorithm** | Simulation | Handle growth with `long long` to avoid overflow. | $O(log n)$ |
| **Missing Number** | Math | Use Arithmetic Series Sum to find the "gap" in $O(1)$ extra space. | $O(n)$ |
| **Repetitions** | Two Pointers | Use a sliding window logic to catch the longest identical streak. | $O(n)$ |
| **Increasing Array** | Greedy | Local optimization: adjust each element to the minimum valid height. | $O(n)$ |
| **Permutations** | Construction | Leverage Parity (Odd/Even) to maintain a distance $\ge 2$. | $O(n)$ |
| **Number Spiral** | Geometry | Layer-based logic: use $max(y, x)$ and parity for coordinate mapping. | $O(1)$ |
| **Two Knights** | Combinatorics | **Inverse Thinking**: Total positions minus "Attack Rectangles" ($2 \times 3$). | $O(1)$ |
| **Two Sets** | Greedy/Math | Sum parity check + Reverse greedy construction from $n$ to $1$. | $O(n)$ |
